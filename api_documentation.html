API Documentation

This project has two API-related parts: auction item data and server health checks. If you are new to coding, the short version is: we fetch JSON, extract a few fields, and display the result in cards.

1) Auction Data (Main Page)

Endpoint used: GET https://api.hypixel.net/v2/skyblock/auctions_ended
Used by: main.html + script.js

Beginner explanation: this endpoint returns ended SkyBlock auctions. For each auction, we decode the item data and show the item name, lore, and final price.

Technical details:

  • auctions[]: list of auction objects
  • item_bytes: compressed/base64 item payload (NBT)
  • price: preferred final price
  • highest_bid_amount: fallback price

2) Minecraft Server Status

Endpoint used: GET https://api.mcsrvstat.us/2/mc.hypixel.net
Used by: status.html + status.js

Beginner explanation: this checks whether the Hypixel Minecraft server is online, and shows player count + version info.

Technical details:

  • online: server online flag
  • players.online and players.max: player counts
  • version: detected server version

Status Labels

  • UP: endpoint/server responded and looks healthy
  • DEGRADED: request succeeded but payload reports partial issue
  • DOWN: request failed or service is offline
